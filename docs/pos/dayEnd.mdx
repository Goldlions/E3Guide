# 日结

## 一、当天没有销售日结步骤

### 步骤1：结束当天营业
- 首先，确保当天的营业已经结束。

### 步骤2：系统提示框
- 系统会弹出提示框。

### 步骤3：确认日结
- 点击“确定”后，系统会提示如下界面：
- 如果点击“取消”，则当天的日结操作即告结束。

## 二、补之前忘记日结（没有销售）

### 步骤1：结束当天营业
- 首先，确保当天的营业已经结束。

### 步骤2：系统提示框
- 系统会弹出提示框。

### 步骤3：选择补录日期
- 系统会再次弹出提示框，此时需要开始补录忘记的营业日期。

### 步骤4：返回日结管理界面
- 返回到**日结** > **日结管理**界面。

### 步骤5：选择补录日期
- 在界面左侧选择需要补录的日期，确保该日期与系统右上角显示的“营业日期”相同。

### 步骤6：提交日结
- 选择好要日结的日期后，点击界面右侧的“提交日结”。

### 步骤7：系统提示框（确定）
- 系统会弹出提示框，点击“确定”按钮。

### 步骤8：系统提示框（取消）
- 如果需要取消日结，点击“取消”按钮。

### 问题一：商店未开始营业
- 如果出现“商店未开始营业”的提示，检查系统右上角的“营业日期”与“要日结的日期”是否一致。
- 如果日期不同，需要先结束系统当前的“营业日期”。
