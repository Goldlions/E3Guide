# 销售退货操作指引手册

## 分销管理 > 销售管理 > 销售退订单

### 第一步：新增销售退订单

#### 基本信息填写
- **单据类型**：根据实际业务需求进行选择。
- **单据日期**：系统默认为当天，如有需要可以修改。
- **业务类型**：根据实际业务需求进行选择。
- **销售项目**：根据实际业务需求进行选择。
- **核算公司**：系统默认为当前账号所属公司。
- **退库存组织**：选择货品要退到的库存组织。
- **退货仓库**：选择货品要退到的仓库。
- **客户**：填写客户相关信息。
- **客户地点、客户联系人、客户联系电话**：系统自动带出。
- 完成基本信息填写后，点击右上角的**保存**按钮。

### 第二步：录入明细

- 准确录入商品明细，并检查确认无误。

### 第三步：提交审核与执行销售退订单

1. **提交销售退订单**：完成明细录入后，提交销售退订单以供审核。
2. **审核销售退订单**：对销售退订单进行审核。
3. **执行销售退订单**：审核通过后，执行销售退订单操作。

#### 执行后操作
执行后会跳转至销售退订单执行页面，进行以下操作：
- **销售退订单-执行**：搜索并执行销售退订单。
- 执行之后会生成对应的销售退单，确认无误后，提交。

### 第四步：入库管理

1. 导航至**仓储中心** > **入库管理** > **销售退货入库扫描工作台**。
2. 输入对应的销售退单单号，按回车键进行搜索。
3. **销售退货入库扫描工作台**：
    - 根据实际情况勾选**次品、退客**。
    - 点击**一键扫描**。
    - 填写**实际到仓日期**，并保存。
    - 点击**扫描完成**。

### 第五步：通知收货与入库审核

1. 返回销售退单页面。
2. **提交**：确保销售退单已提交。
3. **通知收货**：通知相关部门进行收货。
4. **入库审核**：完成入库审核，并确认日期。

---

请按照上述步骤操作，确保每一步都准确无误。如果遇到任何问题或需要进一步的帮助，请随时联系我们。
