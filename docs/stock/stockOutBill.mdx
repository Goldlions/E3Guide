# 调拨出库单（总仓发货至门店仓库）

## 文字操作指引

### 步骤1：创建调拨出库单
1. 首页 → 库存中心 → 库存管理 → 调拨出库单
2. 在调拨出库单页面，点击左上角的**新增**按钮。
3. 填写基本信息：
    - **调出逻辑仓库**：选择公司总仓（例如：XXX-逻辑仓；例：代理商公司-逻辑仓）
    - **调入逻辑仓库**：选择店铺仓库（例如：代理商自营店）
    - 点击右上角的**保存**按钮。

### 步骤2：录入商品明细
4. 在调拨出库单页面，点击**商品明细**，录入商品明细。
5. 提交商品明细后，进行数量审核。

### 步骤3：提交出库单
6. 首页 → 仓储中心 → 出库单
7. 找到对应的单据，点击**提交**。
8. 审核出库单。
9. 匹配通知数。
10. 验收商品。
11. 确认日期（默认为当天）。

### 步骤4：前台（POS）操作
12. 后台跳转至首页 → POS 收银，或直接登录前台：
    [登录前台 POS](https://e3pos.goldlion.com.cn:8443/?#/login-pos)
13. 首页 → 今日待办（右上角） → 门店入库单
14. 点击门店入库单，选择**一键入库**（找到对应的单据）。

## 图文操作指引

### 步骤1：创建调拨出库单
1. 首页 → 库存中心 → 库存管理 → 调拨出库单
2. 调拨出库单页面，点击左上角的**新增**按钮。

### 步骤2：填写基本信息
3. 填写基本信息：
    - **调出逻辑仓库**：选择公司总仓，一般为 XXX-逻辑仓；例：代理商公司-逻辑仓

    - *必填项提示*：填写完调出、调入逻辑仓库后，系统自动带出收获信息。

### 步骤3：录入商品明细
4. 调拨出库单 → 商品明细，录入商品明细。
5. 提交商品明细后，进行数量审核。

### 步骤4：提交出库单
6. 首页 → 仓储中心 → 出库管理 → 出库单
7. 出库单页面，找到对应的单据，点击**提交**。
8. 审核出库单。
9. 匹配通知数。
10. 验收商品。
11. 确认日期（默认为当天）。

### 步骤5：前台（POS）操作
12. 后台跳转至首页 → POS 收银，或直接登录前台：
    [登录前台 POS](https://e3pos.goldlion.com.cn:8443/?#/login-pos)
13. 首页 → 今日待办（右上角） → 门店入库单
14. 门店入库单页面，点击**一键入库**（找到对应的单据）。

请按照上述步骤操作，确保每一步都准确无误。如果有任何疑问或需要进一步的帮助，请及时联系我们。
