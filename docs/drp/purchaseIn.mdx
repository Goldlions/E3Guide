

# 正常货品收货流程

本指南适用于以下两种收货场景：
- 广州总部分发货给代理商（直属经销商）收货
- 代理商发货给经销商收货

## 首页操作

### 打开入库单
您可以通过以下两种方式之一打开入库单：

#### 方式一：搜索栏
- 在首页的搜索栏输入“入库单”。
- 点击搜索结果中名称完全匹配的“入库单”即可打开。

#### 方式二：导航路径
- 依次导航至：**仓储中心** > **入库管理** > **入库单**。

### 匹配通知数
- 在入库单页面，找到对应的入库单。
- 执行匹配通知数的操作。

## 采购单操作

### 打开采购单
您可以通过以下两种方式之一打开采购单：

#### 方式一：首页快捷操作
- 在首页直接选择**采购单**。

#### 方式二：导航路径
- 依次导航至：**供应链管理** > **采购管理** > **采购单**。

### 入库审核与日期确认
- 在采购单页面，找到对应的采购单。
- 执行入库审核。
- 确认日期（一般默认为当天即可）。
